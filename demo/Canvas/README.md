#canvas

> 画布控件: 显示图形元素如线条或文本, 结构化的图形, 用于绘制图形, 创建图形编辑器以及实现自定制的小构件类

| 内置函数         | 说明   |     |
| :--------------- | ------ | --- |
| create_rectangle | 矩形   |     |
| create_oval      | 椭圆   |     |
| create_arc       | 圆弧   |     |
| create_polygon   | 多边形 |     |
| create_line      | 线段   |     |
| create_text      | 文本   |     |
| create_bitmap    | 位图   |     |
| create_image     | 图片   |     |
